# Instructions pour les images

## Images des avatars (section Hero)

J'ai créé un dossier `images/` et modifié le code pour utiliser vos 3 images de closers.

### Étapes à suivre :

1. **Renommez vos 3 images** comme suit :
   - Image 1 (homme musclé) → `avatar1.jpg`
   - Image 2 (homme souriant fond bleu) → `avatar2.jpg`
   - Image 3 (homme avec cigare et champagne) → `avatar3.jpg`

2. **Placez ces images** dans le dossier : `/Users/baptiste/Desktop/Website Theaa/images/`

3. Les images apparaîtront automatiquement dans les petites bulles sous le titre principal.

### Format recommandé :
- Format : JPG ou PNG
- Taille : minimum 100x100px (les images seront redimensionnées à 40x40px)
- Poids : optimisé pour le web (< 100KB par image de préférence)

### Note :
- Les images seront automatiquement arrondies en cercle
- Elles auront une bordure blanche
- Au survol, elles s'agrandiront légèrement
